# WishHelper

Highlights Mirage Wishes in Path of Exile with tier colors and customizable rankings.

<img width="1062" height="618" alt="image" src="https://github.com/user-attachments/assets/c72fe26c-b139-45cc-850b-382f655b03f3" />

## Features

- Draws colored boxes around wishes in the Mirage panel
- S Tier = Purple, A Tier = Green, B Tier = Yellow, C Tier = Orange, D/F Tier = Red
- Displays tier label (S, A, B, C, D, F) on each wish
- Fully customizable tier assignments via settings
- Tooltips show wish descriptions
