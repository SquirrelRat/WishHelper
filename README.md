# WishHelper

Highlights Mirage Wishes in Path of Exile with tier colors and weight-based recommendations.

<img width="1058" height="644" alt="image" src="https://github.com/user-attachments/assets/cf066465-6e0e-4f36-8559-6bebdb3f99c3" />

## Features

- Colored boxes around wishes (S=90-100 Purple, A=80-89 Green, B=70-79 Yellow, C=60-69 Orange, D=0-59 Red)
- Tier labels and "Recommended" indicator for highest weight wish
- Weight sliders (0-100) with auto-calculated tiers
- Hotkey to auto-select recommended wish and confirm
- Reset to Defaults button
